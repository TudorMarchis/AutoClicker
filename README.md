## Quick Start

1. <b>Install dependencies</b>:
	 ```bash
	 pip install -r requirements.txt
	 ```
2. <b>Run BaseClick</b>:
	 ```bash
	 python main.py
	 ```
3. <b>Bind triggers</b> (side mouse or keyboard), set your speed & go!

Settings are saved to <code>%APPDATA%/BaseClick/config.json</code> on Windows.
